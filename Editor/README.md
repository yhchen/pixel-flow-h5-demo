# 像素流 (Pixel Flow) 关卡编辑器使用说明

此目录包含用于批量生成游戏关卡的工具。

## 目录结构

- `gen_levels.py`: 核心生成脚本，基于 Python 编写。
- `README.md`: 本说明文件。

## 如何使用

### 1. 环境准备

确保您的系统中已安装 Python 3 环境。建议使用 `uv` 进行依赖管理（如果脚本后续增加了外部依赖）。

### 2. 生成关卡

在项目根目录或 `Editor` 目录中运行以下命令：

```bash
uv run python Editor/gen_levels.py
```

或者使用标准的 python 命令：

```bash
python3 Editor/gen_levels.py
```

### 3. 说明

- **自动更新**: 脚本运行后，会自动更新项目根目录下的 `mapdata.js` 文件。
- **自定义关卡**: 脚本内部维护了一个 `existing_levels` 列表（前 10 关为手动设计）。您可以修改脚本中的 `TOTAL_LEVELS` 常量来更改生成的总关卡数。
- **难度逻辑**: 脚本实现了“每 15 关增加一种新颜色”的逻辑，确保了游戏难度的阶梯式上升。

## 注意事项

- 在运行生成脚本前，请备份现有的 `mapdata.js`（如果您手动修改过该文件并希望保留）。
- 脚本目前使用的是随机填充算法，生成的关卡保证颜色配比合理，但具体解法可能存在多种。
